# 心愈测 - SCI-90 心理健康评估系统

一款专业的 SCI-90 心理健康在线测试工具，支持手机端与PC端完美适配。

## 产品特点

- 专业可靠：基于标准 SCI-90 量表，包含90道精心设计的题目
- 便捷高效：随时随地通过手机或电脑完成测试
- 隐私安全：数据仅保存在本地浏览器，不会上传到服务器
- 进度保存：支持暂停和继续测试，进度自动保存
- 详细报告：提供总分、因子分、风险等级等多维度分析
- 永久有效：专属分享链接，随时查看历史报告

## 快速开始

### 方式一：直接打开（推荐）

1. 双击打开 `index.html` 文件
2. 浏览器会自动加载应用
3. 点击"开始测试"即可使用

### 方式二：使用本地服务器

如果你安装了 Python，可以在项目目录运行：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器访问：`http://localhost:8000`

### 方式三：使用 WebStorm

1. 用 WebStorm 打开项目文件夹
2. 右键点击 `index.html`
3. 选择 "Open In Browser" 或使用快捷键 `Alt + F2`

## 功能说明

### 测试流程

1. **首页**：查看测试说明和隐私保护政策
2. **测试页面**：
   - 逐题作答，每题5个选项（从"没有"到"严重"）
   - 可随时查看测试进度
   - 支持暂停保存，下次继续
3. **结果报告**：
   - 总分、总均分、阳性项目数
   - 9个维度（因子）的详细分析
   - 风险等级评估
   - 个性化建议和专业帮助指引

### 测试维度

SCI-90 包含以下9个评估维度：

1. **躯体化**：身体不适感
2. **强迫症状**：强迫思维和行为
3. **人际关系敏感**：人际交往困扰
4. **抑郁**：情绪低落、兴趣减退
5. **焦虑**：紧张、不安
6. **敌对**：敌对情绪和行为
7. **恐怖**：恐惧和回避
8. **偏执**：猜疑和多疑
9. **精神病性**：特殊思维体验

### 数据存储

- **本地存储**：测试进度保存在浏览器的 localStorage 中
- **URL分享**：测试结果编码在URL中，便于保存和分享
- **永久有效**：专属链接可随时访问，不依赖服务器

## 技术架构

- **前端框架**：Vue 3（CDN方式引入）
- **数据存储**：localStorage + URL编码
- **响应式设计**：CSS Media Queries
- **浏览器兼容**：支持主流现代浏览器
